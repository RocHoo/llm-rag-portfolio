✨ 核心亮点：

✅ 模块化设计，易于复用与二次开发

✅ 支持主流 LLM API 与本地开源模型

✅ 包含从数据接入到前端交互的完整流程示例

✅ 适合开发者学习、实验与快速搭建 RAG 应用
